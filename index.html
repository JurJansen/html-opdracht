Linux commands

Algemeen handige commands: 


            ls          # Lijst bestanden en mappen
            tree        # Toon mappenstructuur als een boom
            cd          # Verander van map
            pwd         # Toon huidige map
            mkdir       # Maak een nieuwe map
            rm          # Verwijder bestanden of mappen
            cp          # Kopieer bestanden of mappen
            mv          # Verplaats of hernoem bestanden
            cat         # Bekijk inhoud van een bestand
            nano        # Tekstbestand bewerken
            chmod       # Verander bestandsrechten
            chown       # Verander eigenaar van een bestand
            top         # Toon actieve processen
            htop        # Verbeterde top (moet soms geïnstalleerd worden)
            df -h       # Schijfruimte overzicht
            du -h       # Grootte van mappen en bestanden

Debian specific commands: 


            sudo apt update           # Update de lijst met beschikbare packages
            sudo apt upgrade          # Upgrade alle geïnstalleerde packages
            sudo apt install pkg      # Installeer een nieuwe package
            sudo apt remove pkg       # Verwijder een package
            sudo apt autoremove       # Verwijder ongebruikte packages
            sudo apt search pkg       # Zoek naar een package
            sudo apt show pkg         # Bekijk informatie over een package

Arch specific commands:

I use Arch btw.


            sudo pacman -Syu          # Update systeem en packages
            sudo pacman -S pkg        # Installeer een package
            sudo pacman -R pkg        # Verwijder een package
            sudo pacman -Rs pkg       # Verwijder een package + ongebruikte dependencies
            sudo pacman -Ss pkg       # Zoek naar een package
            sudo pacman -Si pkg       # Bekijk info over een package
            sudo pacman -Qi pkg       # Bekijk info over een geïnstalleerde package
