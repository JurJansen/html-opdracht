Linux commands

Klik op de command blocks om ze fullscreen te maken

globale Linux commands

ls                     # laat files en directories in een directory zien
tree                   # laat de files en directories in een tree shape zien (moet vaak nog geinstalleerd worden)
cd [path]              # change directory, laat je van directory veranderen
pwd                    # print working directory, laat zien in welke directory je je bevind
mkdir [name]           # maak een nieuwe directory aan
rm [file]              # verijder files
cp [src] [dest]        # copier een file naar een andere destination
mv [src] [dest]        # verplaatst een file naar een destination
cat [file]             # laat de inhoud van een file zien
nano [file]            # laat je de inhoud van een file aanpassen
chmod [perm] [file]    # laat je de permissions van een file aanpassen
chown [user] [file]    # laat je de eigenaar van een file aanpassen
top                    # laat actieve processen zien
df -h                  # laat disk gebruik zien
du -h [dir]            # laat de size van directories zien

Debian/Ubuntu commands

sudo apt update        # Update packages lijst
sudo apt upgrade       # upgrade geinstalleerde packages
sudo apt install [pkg] # installeert een package
sudo apt remove [pkg]  # verwijderd een package
sudo apt autoremove    # verwijderd ongebruikte packages
sudo apt search [pkg]  # zoeken naar een package
sudo apt show [pkg]    # informatie over een package

Arch Linux commands

sudo pacman -Syu       # Update packages en het systeem zelf
sudo pacman -S [pkg]   # installeert een package
sudo pacman -R [pkg]   # verwijderd een package
sudo pacman -Rs [pkg]  # verwijderd een package en dependencies
sudo pacman -Ss [pkg]  # zoekt voor een package
sudo pacman -Si [pkg]  # laat informatie over een package zien
sudo pacman -Qi [pkg]  # laat geinstalleerde packages zien
